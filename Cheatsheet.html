HTML Cheat Sheet

Never forget your tags again!

Table Tags

Tag Name Desription
<table> Table The wrapper element for all HTML tables
<thead> table header The set of rows defining the column headers
<tbody> table body The set of rows containing the actual table data
<tr> table row The table row container
<td> table data The table data container
<tfoot> table footer The set of rows defining the footer of the table

Table Attributes

Tag Name Description
colspan="" Column span Defines how many columns a td element should span
rowpsan="" Row span Defines how many rows a td element should span
